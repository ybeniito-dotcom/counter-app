📊 Counter App

A simple and interactive counter application built using HTML, CSS, and JavaScript (Web Components). This project demonstrates how to create reusable UI elements and manage state in a clean and modular way.

🔗 Live Demo: https://counter-app-sigma-one.vercel.app/

📂 Repository: https://github.com/ybeniito-dotcom/counter-app

🚀 Features
➕ Increment counter
➖ Decrement counter
🔄 Reset counter
⚡ Fast and lightweight
🧩 Built using reusable Web Components
🎯 Beginner-friendly project
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6+)
Web Components (Custom Elements)
📂 Project Structure
counter-app/
│── index.html          # Main HTML file
│── src/
│   └── counter-app.js  # Web component logic
│── README.md
⚙️ How It Works
The counter logic is handled inside a custom Web Component
Buttons trigger JavaScript functions to:
Increase value
Decrease value
Reset value
UI updates dynamically without page reload
🧑‍💻 Getting Started
1. Clone the repository
git clone https://github.com/ybeniito-dotcom/counter-app.git
2. Navigate to the project folder
cd counter-app
3. Open in browser

Just open index.html in your browser

✏️ Customization
Modify UI → edit index.html
Update logic → edit src/counter-app.js
Add styles → extend CSS inside component or HTML
